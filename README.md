# 📊 Portfólio de Dados — VitaClin

**Análise completa de dados de uma clínica ambulatorial fictícia (VitaClin)**  
Desenvolvido por **Maria Eduarda Albuquerque**

---

## 🏥 Sobre o Projeto

A VitaClin é uma clínica ambulatorial fictícia com 9 unidades de atendimento.
Este portfólio reúne uma análise completa dos dados de atendimentos do período de **janeiro de 2024 a dezembro de 2025**,
cobrindo 5.067 registros e explorando faturamento, volume de atendimentos, perfil de pacientes e oportunidades de negócio.

## 📌 Principais Insights

- 💰 Faturamento total de **R$ 157.936** com ticket médio de **R$ 31,17**
- 🏆 **Filial II–10** tem o maior ticket médio (R$ 52,27) mas opera com apenas 11% do volume da Matriz — maior oportunidade de crescimento
- 🔬 **Exames Laboratoriais** representam 57% dos atendimentos, mas com ticket baixo de R$ 19 — motor de captação, não de margem
- 📡 **Ultrassonografia** gera 15,9% da receita com apenas 4,8% dos atendimentos — alto potencial de expansão
- 📈 Recuperação expressiva no segundo semestre de 2025, com **julho crescendo 137%** em relação ao mesmo período de 2024
